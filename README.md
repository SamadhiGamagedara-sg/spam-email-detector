

This project is a machine learning-based web application that classifies emails as Spam or Not Spam.

Technoloies used
- Python
- Scikit-learn
- Pandas
- Streamlit

Features 
- Input any email message
- Predicts whether it is Spam or Not Spam
- Simple and interactive UI

- How to run 
```bash
pip install -r requirements.txt
streamlit run spam_app.py
